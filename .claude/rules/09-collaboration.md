# Collaboration and Tooling

## Version Control
- Use Git effectively
- Follow established branching strategies (Gitflow, GitHub Flow)
- Write clear and concise commit messages
- Utilize pull/merge requests for code review

## Collaboration Tools
- Use project management tools (Jira, Trello)
- Use communication tools (Slack, Teams)
- Use documentation tools (Confluence, Notion)

## Development Environment
- Strive for consistent and reproducible development environments
- Use tools like Docker, Nix, or standardized setup scripts
