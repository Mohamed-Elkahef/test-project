Responsive Breakpoints Summary

Mobile (320px - 767px):
  • 2x2 grid for stat cards
  • List view for status breakdown (no chart)
  • Card-based layout for recent orders
  • Stacked sections, full-width components
  • Abbreviated labels for space efficiency
Tablet (768px - 1023px):
  • 4 stat cards in single row
  • 2-column grid for status items
  • Table layout for recent orders (condensed columns)
  • Sections remain full-width
  • More padding and spacing than mobile
Desktop (1024px+):
  • 4 stat cards in single row with larger padding
  • 2-column grid for status section and quick stats
  • Donut chart visualization with legend
  • Full table for recent orders with all columns
  • Maximum content width of 1400px, centered
